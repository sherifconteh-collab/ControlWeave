# 🗺️ Feature Roadmap

This document outlines planned features and improvements for ControlWeave.

> **Note**: Roadmap items are subject to change. Subscribe to release notes for official announcements.

## Recently Shipped

- [x] **AI Evidence Suggestions** *(v2.3.0)*: AI scans connected integrations (Splunk), maps data to framework controls, and suggests evidence for user approval — no manual hunting required
- [x] **Auto-Evidence Collection** *(v2.3.0)*: Schedule-based and on-demand evidence collection from Splunk via collection rules

## Q2 2026 (Planned)

### High Priority
- [ ] **Expanded Auto-Evidence Sources**: Connect to cloud providers (AWS CloudTrail, Azure, GCP) and SIEM tools (Microsoft Sentinel, CrowdStrike) for auto-evidence collection
- [ ] **Continuous Control Monitoring (CCM)**: Real-time control effectiveness monitoring with automated alerts
- [ ] **Advanced Reporting Engine**: Customizable report templates with drag-and-drop builder
- [ ] **FedRAMP Package Automation**: Automated SSP, SAR, and POA&M generation for FedRAMP packages

### Medium Priority
- [ ] **Multi-Organization Support**: Manage multiple organizations from a single account
- [ ] **Advanced Crosswalk Editor**: Visual crosswalk mapping tool with custom framework support
- [ ] **Mobile App**: iOS and Android app for on-the-go compliance management
- [ ] **Slack/Teams Integration**: Compliance alerts and AI Copilot accessible via chat

## Q3 2026 (Planned)

- [ ] **GRC Workflow Automation**: No-code workflow builder for compliance processes
- [ ] **Enhanced TPRM**: Automated vendor security questionnaire scoring
- [ ] **Custom Framework Builder**: Build and manage proprietary compliance frameworks
- [ ] **Advanced Analytics**: Predictive compliance risk modeling

## Q4 2026 (Research Phase)

- [ ] **AI-Native Evidence Review**: Automated evidence sufficiency analysis using computer vision
- [ ] **Regulatory Change Detection**: Automatic alerts when regulatory requirements change
- [ ] **Compliance Marketplace**: Exchange policies, procedures, and controls with the community
