# 📋 NIST SP 800-53 Guide

Using ControlWeave with the NIST Special Publication 800-53 framework.

## Overview

NIST SP 800-53 provides a catalog of security and privacy controls for federal information systems and organizations. ControlWeave implements the Rev 5 control catalog with all 20 control families and 1,000+ controls and control enhancements.

## Impact Baselines

NIST 800-53 defines three security baselines:

| Baseline | Use Case |
|----------|----------|
| **Low** | Minimal security requirements, low-impact systems |
| **Moderate** | Standard security, moderate-impact systems |
| **High** | Enhanced security, high-impact systems (FedRAMP High) |

## Using NIST 800-53 in ControlWeave

### Activate the Framework
1. Go to **Frameworks** → **Add Framework**
2. Select **NIST 800-53 Rev 5**
3. Choose your impact baseline (Low/Moderate/High)
4. Click **Activate**

### Tailor Controls
Mark controls as Not Applicable if they don't apply to your environment:
1. Open the control
2. Click **Mark as N/A**
3. Provide a tailoring justification

### Crosswalk Mappings
NIST 800-53 crosswalks available in ControlWeave:
- NIST 800-53 ↔ ISO 27001:2022
- NIST 800-53 ↔ SOC 2 Trust Services Criteria
- NIST 800-53 ↔ CMMC 2.0
- NIST 800-53 ↔ FedRAMP
