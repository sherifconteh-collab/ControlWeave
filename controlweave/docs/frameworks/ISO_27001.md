# 🔒 ISO 27001:2022 Guide

Using ControlWeave with the ISO/IEC 27001:2022 information security management standard.

## Overview

ISO 27001:2022 is the international standard for information security management systems (ISMS). ControlWeave implements the 2022 version with all 4 organizational controls clauses and 93 Annex A controls across 4 themes.

## Control Themes (Annex A)

| Theme | Controls | Description |
|-------|----------|-------------|
| **Organizational** | 37 | Policies, roles, supplier relationships |
| **People** | 8 | HR security, awareness, remote work |
| **Physical** | 14 | Physical security, clear desk |
| **Technological** | 34 | Access control, cryptography, networks |

## Key Clauses

The ISO 27001 mandatory clauses that must be addressed:

| Clause | Title |
|--------|-------|
| 4 | Context of the Organization |
| 5 | Leadership |
| 6 | Planning |
| 7 | Support |
| 8 | Operation |
| 9 | Performance Evaluation |
| 10 | Improvement |

## Using ISO 27001 in ControlWeave

### Activate the Framework
1. Go to **Frameworks** → **Add Framework**
2. Select **ISO 27001:2022**
3. Click **Activate**

### Statement of Applicability (SoA)
The Statement of Applicability documents which Annex A controls are included and why:
1. Go to **Frameworks** → **ISO 27001** → **Statement of Applicability**
2. For each control, mark as **Included** or **Excluded** with justification
3. Export the SoA as a PDF for audit purposes

### Crosswalk Mappings
ISO 27001 crosswalks available in ControlWeave:
- ISO 27001 ↔ NIST 800-53 Rev 5
- ISO 27001 ↔ SOC 2
- ISO 27001 ↔ GDPR
