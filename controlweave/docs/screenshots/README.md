# Screenshots Guide

This directory contains screenshots for the user guide documentation.

## Screenshot Guidelines

### Naming Convention
Screenshots should be named using this format:
```
<feature>-<action>-<number>.png
```

Examples:
- `dashboard-overview-01.png`
- `frameworks-activate-01.png`
- `controls-edit-implementation-01.png`
- `ai-copilot-chat-01.png`

### File Format
- **Format**: PNG (for UI screenshots)
- **Resolution**: 1920x1080 or consistent size
- **Max File Size**: 500KB per screenshot (optimized)

### Viewing
Screenshots are referenced in documentation using:
```markdown
![Alt text](../screenshots/filename.png)
```
