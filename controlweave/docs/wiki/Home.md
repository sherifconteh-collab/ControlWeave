## 🎯 Feature Availability

ControlWeaver is fully open source with no tier gating — Frameworks, unlimited CMDB assets, all assessment depths, AI Copilot, evidence management, Auditor Workspace, Custom Dashboards, AI Monitoring/Data Governance/Vendor Risk Dashboards, Predictive Risk Scoring, Regulatory Impact Analysis, Smart Remediation Plans, Threat Intelligence Feeds, Vendor Security Monitoring, Regulatory News Feeds, SBOM/AIBOM, Vulnerability Management, POA&M Tracking, SSO/SAML, and SIEM Integration are all available to every authenticated user.

## 💡 Key Features

### Multi-Framework Support
- 15+ compliance frameworks supported
- 500+ controls with crosswalk mappings
- Automatic control overlap detection
- Framework selection per organization

### AI-Powered Features
- BYOK (Bring Your Own Key) for multiple LLM providers
- 25+ AI analysis features
- AI Copilot with org-aware context
- Gap analysis and compliance forecasting

### Assessment & Audit
- 186 assessment procedures across frameworks
- Three assessment depths: Basic, Focused, Comprehensive
- NIST-standard outcomes
- External auditor workspace with findings tracking

### Asset & Vulnerability Management
- Configuration Management Database (CMDB)
- SBOM/AIBOM support
- Vulnerability tracking with CVSS scoring
- POA&M (Plan of Action & Milestones)
