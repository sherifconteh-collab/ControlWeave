# ControlWeave MCP Marketplace Listing Pack (Tier-Aligned)

## Listing Title
ControlWeave MCP: Tier-Aware AI Compliance Copilot

## Short Description
Secure MCP tools for compliance operations, with capabilities automatically aligned to your ControlWeave subscription tier.

## Long Description
ControlWeave MCP connects AI assistants to real compliance workflows in ControlWeave. Teams can query posture, manage controls, link evidence, and execute governance workflows using permission-aware MCP tools with auditability built in.

Unlike standalone MCP products, ControlWeave MCP follows your existing ControlWeave tier entitlements. As your subscription tier changes, your MCP capabilities expand automatically.

## Key Value Points
- Tier-aligned access model (Free, Starter, Professional, Enterprise, Utilities)
- Secure-by-default MCP server (rate limits, validation, audit logs)
- Role and permission-aware actions
- Managed or self-hosted deployment
- Fast setup with connection smoke testing
