# 🎯 Quick Wins Guide

## 5-Minute Quick Wins

### Win #1: Activate Your Core Framework

**Time**: 2 minutes  
**Impact**: Instant access to 50-300 controls

**Steps**:
1. Go to **Frameworks**
2. Click **Activate** on your primary framework (SOC 2, NIST 800-53, ISO 27001)
3. Done!

**Value**: You now have a structured compliance roadmap.

---

### Win #2: Run AI Gap Analysis

**Time**: 3 minutes  
**Impact**: Instant compliance assessment  
**Requires**: AI configured

**Steps**:
1. Go to **AI Analysis**
2. Click **Gap Analysis**
3. Wait 30 seconds
4. Review results

![AI Gap Analysis results showing missing controls](../screenshots/ai-gap-analysis-quick-01.png)

**Value**: Know exactly what you're missing and where to focus.

## 15-Minute Quick Wins

### Win #3: Implement 3 Policy Controls

**Time**: 15 minutes  
**Impact**: 3-10% compliance boost (depending on framework)

**Why These**: Policy controls are the easiest - just upload documents you likely already have.

**Target Controls**:
1. **AC-1** (NIST) / **A.5.1** (ISO) / **CC1.1** (SOC 2) - Information Security Policy
2. **AT-1** (NIST) / **A.6.3** (ISO) / **CC1.4** (SOC 2) - Security Awareness Training
3. **IR-1** (NIST) / **A.16.1** (ISO) / **CC9.1** (SOC 2) - Incident Response

**Steps for Each Control**:
1. Navigate to **Controls**
2. Search for control (e.g., "AC-1")
3. Click control → **Edit Implementation**
4. Set Status: **Implemented**
5. Add Owner: Yourself
6. Add Notes: "Company Information Security Policy v2.0"
7. **Add Evidence**: Upload your policy document
8. **Save**

**Time per Control**: ~5 minutes

**Value**: 
- Quick compliance percentage boost
- Foundation policies documented
- Evidence library started

---

### Win #4: Leverage Crosswalks

**Time**: 10 minutes  
**Impact**: Discover you're more compliant than you thought

**If You Have Multiple Frameworks**:

1. Go to **AI Analysis** → **Crosswalk Optimizer**
2. Click **Run Analysis**
3. AI identifies overlapping controls
4. Review recommendations

**Example Results**:
```
Your NIST 800-53 AC-2 implementation also satisfies:
✓ ISO 27001 A.9.2.1
✓ SOC 2 CC6.1
✓ HIPAA 164.308(a)(3)(i)

Result: 1 implementation = 4 controls satisfied!
```

**Value**: Instantly increase compliance % across multiple frameworks.

---

### Win #5: Create Your First Assessment

**Time**: 10 minutes  
**Impact**: Validate your first control

**Steps**:
1. Go to **Assessments** → **New Assessment**
2. Select a control you just implemented (AC-1, AT-1, or IR-1)
3. Select **Self-Assessment**, **Basic** depth
4. Set Due Date: Today
5. Click **Create**
6. Review assessment procedures
7. Mark each as **Satisfied** (if your evidence supports it)
8. Add notes: "Policy reviewed, approved by management, all staff notified"
9. Click **Save Results**

![Assessment completed showing Satisfied results](../screenshots/assessment-completed-quick-01.png)

**Value**:
- First control verified
- Assessment process learned
- Control health score improved

## 30-Minute Quick Wins

### Win #6: Build Your Evidence Library

**Time**: 20 minutes  
**Impact**: Foundation for future assessments

**Gather Common Evidence** (you probably have these):
1. Information Security Policy
2. Acceptable Use Policy
3. Incident Response Plan
4. Business Continuity Plan
5. Employee handbook (with security section)
6. Training materials/records
7. Recent vulnerability scan report
8. System architecture diagram
9. Access control matrix
10. Password policy

**Upload All at Once**:
1. Go to **Evidence**
2. Click **Bulk Upload** (drag multiple files)
3. Add tags to each (policy, procedure, report, etc.)
4. Click **Upload All**

**Value**: Evidence ready for future control implementations.

---

### Win #7: Set Up Team

**Time**: 10 minutes  
**Impact**: Enable collaboration

**Steps**:
1. Go to **Settings** → **Users**
2. Click **Invite User**
3. Add key team members:
   - IT Manager → Admin role
   - Security Engineer → Manager role
   - Compliance Officer → Manager role
   - Department Heads → Analyst role
4. Assign controls to appropriate owners
   - Go to **Controls**
   - Filter by family (e.g., **AC** for IT, **AT** for HR)
   - Bulk assign to appropriate owner

**Value**: Distributed work, shared responsibility.

---

### Win #8: Create Dashboard View

**Time**: 5 minutes  
**Impact**: Instant executive visibility

1. Go to **Custom Dashboards**
2. Click **Create Dashboard**
3. Name: "Executive Overview"
4. Add widgets:
   - Compliance Overview
   - Framework Progress
   - Priority Actions
   - Compliance Trend
5. **Save**
6. Share link with executives

![Custom executive dashboard](../screenshots/dashboard-custom-executive-01.png)

**Value**: Leadership visibility, stakeholder buy-in.

## Quick Win Strategies

### Strategy #1: Document What You Already Do

Most organizations already do security things - they just haven't documented it.

**Quick Documentation**:
1. Use AI Copilot: "Generate an access control policy based on our current practices"
2. Review and customize
3. Get approval
4. Upload as evidence
5. Mark controls as Implemented

**Time**: 10-15 minutes per policy  
**Impact**: Multiple controls satisfied

---

### Strategy #2: Low-Hanging Fruit First

Target controls that are:
- ✅ Already done (just need documentation)
- ✅ Easy to implement (policies, not technology)
- ✅ High-value (crosswalk to multiple frameworks)

**Use AI to Find These**:
Ask Copilot: "What are the quickest controls I can implement today?"

---

### Strategy #3: Leverage Existing Documentation

**Common Documents Everyone Has**:
- Employee handbook → AT-1, PS-1
- IT policies → Multiple AC, IA controls
- Backup procedures → CP-9
- Disaster recovery plan → CP-2, CP-10
- Change management process → CM-3
- Incident response runbook → IR-4, IR-5

**Action**: Link these to relevant controls as evidence.
