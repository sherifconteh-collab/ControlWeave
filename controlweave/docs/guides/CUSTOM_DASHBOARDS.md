# 🎨 Custom Dashboards Guide

Build personalized compliance dashboards tailored to your role and workflow.

## Overview

Custom Dashboards let you create role-specific views combining the metrics, charts, and data that matter most to you.

## Accessing Custom Dashboards

Navigate to **Dashboards** → **My Dashboards** → **New Dashboard**

## Creating a Dashboard

1. Click **+ New Dashboard**
2. Enter a **Dashboard Name** (e.g., "CISO Weekly View")
3. Select a **Layout**: 1-column, 2-column, or 3-column grid
4. Click **Create**

## Available Widgets

Drag-and-drop widgets onto your dashboard canvas:

### Compliance Widgets
| Widget | Description |
|--------|-------------|
| **Overall Score** | Single compliance percentage gauge |
| **Framework Scores** | Side-by-side framework comparison |
| **Control Status Ring** | Donut chart of control implementation |
| **Compliance Trend Line** | Time-series compliance graph |

### Evidence & Assessment Widgets
| Widget | Description |
|--------|-------------|
| **Evidence Expiry Calendar** | Upcoming evidence renewals |
| **Assessment Due Dates** | Scheduled and overdue assessments |
| **Recent Evidence** | Latest uploaded evidence items |

### Vulnerability Widgets
| Widget | Description |
|--------|-------------|
| **Open Vulnerability Count** | Count by severity |
| **MTTR Gauge** | Mean time to remediation |
| **Vulnerability Trend** | Open/closed over time |

### AI & Insights Widgets
| Widget | Description |
|--------|-------------|
| **AI Analysis Results** | Latest AI analysis summary |
| **Risk Score** | AI-calculated risk rating |
| **Compliance Forecast** | Projected compliance at 30/60/90 days |

### Custom Text/Links Widgets
| Widget | Description |
|--------|-------------|
| **Markdown Note** | Free-text with markdown formatting |
| **Quick Links** | Customizable bookmark links |

## Configuring Widgets

1. Click the ⚙️ gear icon on any widget
2. Configure widget-specific settings:
   - **Framework filter**: Scope to specific frameworks
   - **Date range**: Historical window
   - **Thresholds**: Set warning/critical levels
3. Click **Apply**

## Sharing Dashboards

Share dashboards with your team:

1. Open the dashboard
2. Click **Share** → **Share with Organization**
3. Set permission: **View Only** or **Edit**
4. Users can find shared dashboards under **Dashboards** → **Shared**

## Dashboard Templates

Start with pre-built templates for common roles:
- **Executive Summary**: High-level KPIs for leadership
- **Compliance Manager**: Day-to-day compliance tracking
- **Security Analyst**: Vulnerability and CMDB focused
- **Auditor View**: Assessment and evidence status
