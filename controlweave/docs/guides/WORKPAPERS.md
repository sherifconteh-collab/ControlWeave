# 📄 Workpapers Guide

Create and manage audit workpapers within ControlWeave's Auditor Workspace.

## Overview

Workpapers are the formal documentation that auditors prepare to support their conclusions and findings. ControlWeave provides a structured workpaper system linked directly to controls and evidence.

## Creating a Workpaper

1. Navigate to **Auditor Workspace** → **Engagements** → select an engagement
2. Click **Workpapers** → **New Workpaper**
3. Fill in the workpaper details:
   - **Title**: Descriptive name for the workpaper
   - **Control Reference**: Link to the control being tested
   - **Procedure**: The audit procedure performed
   - **Evidence**: Attach or reference collected evidence
4. Click **Save**

## Workpaper Structure

Each workpaper contains:

| Section | Description |
|---------|-------------|
| **Header** | Engagement, control reference, auditor, date |
| **Objective** | What the test is designed to verify |
| **Procedure** | Steps performed during testing |
| **Evidence** | Documents and artifacts reviewed |
| **Results** | Observations from testing |
| **Conclusion** | Auditor's assessment (Satisfactory / Unsatisfactory) |

## Workpaper Status Tracking

| Status | Meaning |
|--------|---------|
| **Draft** | In preparation, not yet reviewed |
| **In Review** | Submitted for supervisor review |
| **Approved** | Reviewed and approved |
| **Requires Revision** | Returned for corrections |

## Evidence Linking

Link evidence directly from ControlWeave's Evidence Library:

1. Open a workpaper
2. Click **Attach Evidence**
3. Search or browse the evidence library
4. Select relevant files and click **Attach**
